# 📊 Model Using K-Means

A beginner-friendly Machine Learning project implementing the **K-Means Clustering Algorithm** using Python.  
This project demonstrates how unsupervised learning groups similar data points into clusters without predefined labels.

---

# 🧠 About K-Means Algorithm

K-Means is an **unsupervised machine learning algorithm** used for clustering data into distinct groups.

It automatically identifies patterns in the data without predefined labels, organizing data points into **K clusters** based on their similarities.

This makes it useful for discovering hidden structures in large datasets.
